# Precptron-ANN-CNN-Mnist-
Handwritten Digit Recognition using the MNIST dataset and Machine Learning. This project trains a classification model to identify handwritten digits (0–9) from grayscale images. It includes data preprocessing, model training, performance evaluation, and prediction of unseen digit images using Python and Scikit-learn/Deep Learning techniques.

# Handwritten Digit Recognition using MNIST Dataset

## Overview

This project implements handwritten digit recognition using the MNIST dataset and compares the performance of three neural network approaches: Perceptron, Artificial Neural Network (ANN), and Convolutional Neural Network (CNN). The models are trained to classify grayscale images of handwritten digits (0–9).

## Features

* Image preprocessing and normalization
* Digit classification using multiple neural network architectures
* Performance comparison across models
* Prediction of unseen handwritten digit images

## Technologies Used

* Python
* NumPy
* Pandas
* TensorFlow / Keras
* Matplotlib

## Models Implemented

1. Perceptron
2. Artificial Neural Network (ANN)
3. Convolutional Neural Network (CNN)

## Project Workflow

1. Load MNIST dataset
2. Preprocess and normalize images
3. Train Perceptron, ANN, and CNN models
4. Evaluate model performance
5. Compare classification results

## Results

The CNN model achieved the best performance by extracting spatial image features effectively compared to Perceptron and ANN.

## Applications

* Handwritten digit recognition
* Image classification
* Optical character recognition (OCR)
* Computer vision tasks
